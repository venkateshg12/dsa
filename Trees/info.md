Binary Tree:
    -- A Binary Tree is a fundamental data structure in computer science where each node can have at most two children.

Nodes: 
    -- Each node in a binary tree contains a piece of data, often referred to as the node’s value or key. This node also 
        contains references and pointers to its left and right children so that we can traverse from one node to another  
        in a hierarchical order.

Root Node: 
    -- Root Node is the topmost node of a binary tree from which all other nodes stem out. This serves as the entry point 
        for traversing the tree structure.

Leaf Node:
    -- Leaf Nodes are the nodes that do not have any children. These nodes lie on the outermost ends of the tree branches 
       and are the terminal points of the traversal.


Types of Binary Tree

Full Binary Tree OR (Strict Binary Tree):
    -- Every node as 0 or 2 children.

Perfect Binary Tree:
    -- All levels are completely filled.

Complete Binary Tree:
    -- All levels filled except possibly the last, filled from left to right.

Skewed Binary Tree:
    -- All nodes go in one direction.
